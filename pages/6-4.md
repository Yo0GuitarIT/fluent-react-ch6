# Hydrating

目標是將事件監聽和其他 JavaScript 功能附加到靜態的 HTML。

```jsx
import { hydrateRoot } from "react-dom/client";
import App from "./app.jsx";

// 客戶端水合 (Hydration)
hydrateRoot(document.getElementById("root"), <App />);
```

-> NOTE:  此時 APP 才真正具備互動功能。
